# cmdavis25.github.io
Chris Davis's GitHub Website

## Design Intent
The primary intent of this website is to serve as a professional landing page for potential employers who would be looking to hire a Data Professional - whether it's a Data Analyst, Data Scientist, or Data Engineer. The site needs to be usable on both mobile and desktop browsers.
